# Telegram Bot Project

## Структура проекта
- `bot.py` - основной файл бота
- `config.py` - конфигурационный файл
- `requirements.txt` - зависимости проекта
- `sync.sh` - скрипт для синхронизации с сервером

## Инструкции по работе

### Синхронизация с сервером
```bash
# Синхронизировать файлы с сервера
./sync.sh from

# Синхронизировать файлы на сервер
./sync.sh to

# Перезапустить бота на сервере
./sync.sh restart

# Синхронизировать и перезапустить бота
./sync.sh deploy
```

### Проверка работы бота
```bash
# Просмотр логов
ssh selectel 'tail -f /var/www/telegram_bot/bot.log'

# Проверка статуса процесса
ssh selectel 'ps aux | grep bot.py'
```

### Важные замечания
1. Файл `.env` с секретными ключами не синхронизируется
2. При деплое бот автоматически перезапускается
3. Логи сохраняются в `bot.log` на сервере

## Текущий статус проекта
- Бот размещен на сервере Selectel
- Путь на сервере: `/var/www/telegram_bot/`
- Поддерживается синхронизация между локальной и серверной версиями
- Настроен автоматический перезапуск при деплое
